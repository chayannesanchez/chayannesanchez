# Chayanne Alfonso Sánchez García

QA Engineer | QA Functional Analyst | Software Test Analyst | Manual Tester | Web Testing

---

Ingeniero de Sistemas con experiencia como QA Funcional y Software Test Analyst, especializado en pruebas funcionales, pruebas de regresión, validación de requerimientos y aseguramiento de la calidad del software en entornos ágiles. Me enfoco en diseñar y ejecutar casos de prueba, documentar resultados, dar seguimiento a incidencias y validar criterios de aceptación para entregar productos confiables, estables y alineados con las necesidades del negocio.

He trabajado en aplicaciones web aplicando Scrum y Kanban, usando herramientas como Jira, TestRail, Postman, Git/GitHub, SQL, Python, Docker y servicios cloud. Me caracterizo por el pensamiento analítico, la atención al detalle, la comunicación efectiva, el trabajo colaborativo y la mejora continua.
